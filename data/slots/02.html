Используем компонент-шаблон 'main-content' для вывода данных компонентов 'first-component' и 'second-component'
Содержимое этих компонентов - кнопка, окошко для вывода контента загруженных файлов и футер, куда выводится сообщение о получении данных или ошибке
Компонент 'main-content' должен быть дочерним для 'first-component' и 'second-component', т.е. внутри каждого из этих компонентов задаем свойство components

components: {
    'main-content': MainContent
}

Внутри 'main-content' определяем плейсхолдеры для контента родительских компонентов 'first-component' и 'second-component' с помощью именованных слотов 

const MainContent = ( 'main-content', {
    template: `
        <div>
          <nav>
            <slot name="button"></slot>
          </nav>
          <slot name="content"></slot>
          <slot name="footer"></slot>
            
        </div>
      `
})

Теперь привязываем контент наших компонентов 'first-component' и 'second-component' к плейсхолдерам в 'main-content' с помощью атрибутов
slot = "button",
slot="content" и
slot="footer"

template: `
    <main-content>
        <button @click="thisClickHandler" 
                v-if = "dataIsReady"
                slot = "button">
          first
        </button>
        <div class="demo"
              slot="content"
              v-html="mess"
              v-if="contentIsVisible">
        </div>
        <div slot="footer"
              v-html="messageToShow">
        </div>
    </main-content>
`

Теперь стилизуйте контент (слоты) внутри 'main-content'
